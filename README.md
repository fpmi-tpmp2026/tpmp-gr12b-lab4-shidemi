# Tourist Bureau Console App

## Project Name

Tourist Bureau Console App — консольное приложение для учета работы туристического бюро по варианту 6 лабораторной работы №4.

## Description

Приложение хранит сведения об экскурсионных маршрутах, автобусах, закрепленных экипажах и выполненных рейсах. Пользователь проходит аутентификацию и получает права администратора или члена экипажа. Администратор может добавлять рейсы, получать отчеты по автобусам, маршрутам и начислениям экипажам. Член экипажа может просматривать свои данные и сумму начислений на указанную дату.

## Installation

1. Установить компилятор C++17 и SQLite:
   ```bash
   sudo apt-get install g++ sqlite3 libsqlite3-dev
   ```
2. Склонировать репозиторий:
   ```bash
   git clone https://github.com/fpmi-tpmp2026/tpmp-gr12b-lab4-shidemi.git
   cd tpmp-gr12b-lab4-shidemi
   ```
3. Собрать проект:
   ```bash
   make all
   ```
4. Запустить тесты:
   ```bash
   make test
   ```

## Usage

Запуск приложения:

```bash
make run
```

Тестовые пользователи:

| Логин | Пароль | Роль |
| --- | --- | --- |
| `admin` | `admin` | администратор |
| `ivanov` | `crew123` | член экипажа |
| `orlova` | `crew123` | член экипажа |
| `smirnov` | `crew123` | член экипажа |

Основные пункты меню позволяют вывести рейсы указанного автобуса за период, получить сводку по автобусу, рассчитать начисления экипажам, найти самый дорогой маршрут, найти автобус с максимальным пробегом, добавить новый рейс, изменить количество пассажиров и удалить рейс.
