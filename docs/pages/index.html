Туристическое бюро

Консольное C++/SQLite приложение для лабораторной работы №4, вариант 6.

Функциональность

Приложение выполняет аутентификацию, хранит маршруты, автобусы, экипажи и рейсы, формирует отчеты и рассчитывает начисления экипажам.
