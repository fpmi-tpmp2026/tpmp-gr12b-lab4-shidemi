# Глоссарий

| Термин | Описание |
| --- | --- |
| Туристическое бюро | Организация, выполняющая экскурсионные рейсы по маршрутам. |
| Экскурсионный маршрут | Направление поездки с названием, начальным и конечным пунктом, протяженностью. |
| Автобус | Транспортное средство, закрепленное за экипажем и используемое в рейсах. |
| Экипаж | Три сотрудника, постоянно закрепленные за одним автобусом. |
| Член экипажа | Сотрудник с табельным номером, стажем, категорией и адресом. |
| Рейс | Факт выполнения экскурсии по маршруту с датами, пассажирами и стоимостью билета. |
| Начисление | Денежная сумма, равная проценту от дохода рейсов автобуса экипажа. |
