# Индивидуальный проект

Исполнитель: Сафиулин Артём, группа 12.

Индивидуальная часть лабораторной работы описывает проектирование требований и UML-спецификации. Для текущего репозитория основная реализация выполнена по командному варианту 6 «Туристическое бюро».

## Структура

- Постановка задачи.
- Диаграмма вариантов использования.
- Диаграммы деятельности.
- Диаграмма классов.
- Диаграммы последовательности.
- Диаграмма компонентов.
